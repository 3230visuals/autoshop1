# Design System: Auto Shop Mobile App
**Project ID:** projects/445351047389082100

## 1. Visual Theme & Atmosphere
- **Vibe:** Modern, professional, automotive, sleek.
- **Theme:** Dark mode by default, high contrast.
- **Experience:** Mobile-first, app-like.

## 2. Color Palette
- **Background:** `bg-background-dark` (#0a0b14) / `bg-zinc-950`
- **Primary:** `text-primary` (#f27f0d) - Orange accent
- **Secondary:** `text-slate-100` - White text
- **Success:** `text-primary-success` (#0df20d)
- **Error:** `text-red-500`

## 3. Typography
- **Headings:** `font-space` (Space Grotesk)
- **Body:** `font-display` (Inter)

## 4. Component Stylings
- **Cards:** Rounded (`rounded-xl`), dark (`bg-card-dark`), subtle border (`border-white/10`).
- **Buttons:** Primary orange with rounded corners, hover effects.
- **Inputs:** Dark background, light text, focused orange border.
